## Setup


### installation


- 始めにインストールした人 = 共有する人
```sh
> npm install --save-dev discord.js
```

- 共有されたものをそのままインストールする人
```sh
> npm install 
```

### gitignore

- `node_modules/` と token が含まれる `config.json` を無視する 


```.gitignore
# Dependency directories
node_modules/


# token

/config.json
```
